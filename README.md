## Usage

To generate HTML and PDF outputs from QMD files:

``` sh
quarto render hw1.qmd --to html
quarto render hw1.qmd --to pdf
```

Replace `hw1.qmd` with the desired file.

## Requirements

-   Quarto
-   R / Python (depending on the analysis)
